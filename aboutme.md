# 관심사

- **Mechanistic Interpretability**

    모델의 작동을 연산 단위로 환원하여 설명하려는 시도를 mechanistic interpretability 라 합니다. Olah et al. (2020) 은 mechanistic interpretability 의 연구 목표로 크게 3가지 질문을 제시합니다.

  1. Feature : 딥러닝 모델이 추출한 feature vector 의 의미는 무엇인가?
  2. Circuit : 딥러닝 모델이 만들어낸 각 feature vector 들은 어떻게 상호작용 하는가?
  3. Universality : 딥러닝 모델 구조와 관계 없이 발견되는 feature 와 circuit 이 있는가?

    저는 위의 3가지 질문을 정보이론, 정보기하학 그리고 인과 추론의 관점에서 답하는 것에 관심 있습니다.
